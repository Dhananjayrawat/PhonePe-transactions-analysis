Project Overview
Analysis examines PhonePe digital payment transactions across India, exploring usage patterns at state and district levels from 2018-2021.

Key Research Questions
How do people across India use PhonePe?
How do transactions vary by state, district, year, and quarter?
How do demographics influence transactions?
How do device types and transaction types impact usage?
Is the data consistent and reliable?

Dataset Summary
36 States/Union Territories covered
732+ Districts analyzed
Data span: 2018-2021 (quarterly data)
Minimal missing values (< 0.3%), ensuring data reliability

Data Fields Analyzed:
Transactions count
Transaction amounts (INR)
Average Transaction Value (ATV)
Registered users
App opens
Device brands/types
District demographics (population, area, density)

Key Findings
Geographic Distribution:
Most active state: Uttar Pradesh (75 districts)
Top performing states: Maharashtra, Karnataka, and Andhra Pradesh show the highest transaction volumes
Clear growth trajectory from 2018-2021 across most states
